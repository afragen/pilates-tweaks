== Description ==

This plugin adds iOS specific CSS and trademarking for Gyrotonics&reg;. This is an add-on for Janna's White Lotus Gyrotonic website. This original code by me.

== Installation ==

1. Upload the entire `/pilates-tweaks/` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin.
